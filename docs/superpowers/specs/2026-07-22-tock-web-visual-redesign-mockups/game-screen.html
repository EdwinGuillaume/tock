Layout figé

  • Barre haute : tour (puce couleur qui bobbe + texte) + pastilles Pioche/Défausse.
  • Journal : une ligne « dernière action » + « ▾ » pour déplier l'historique.
  • Plateau : héros, centré, le plus grand.
  • Hint : chip discret (opacité réduite, fin cadre, fond très léger).
  • Main : cartes rang + enseigne, éventail, sélection qui bondit, injouables grisées.
