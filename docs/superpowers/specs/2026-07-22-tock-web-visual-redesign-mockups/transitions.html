Ce que tu sens

Fin de partie : fondu + accélération (~0,3 s). Dans la vraie appli, les confettis partiront une fois l'écran posé.

Passe le téléphone : le voile devient opaque très vite (~0,16 s) pour couvrir la main du joueur précédent, puis le contenu s'installe — pas de transparence qui laisse fuiter l'info.

Retour au jeu : fondu classique.

Direction validée : fondu + accélération partout, couverture opaque rapide pour le passage de joueur, quasi-instantané en prefers-reduced-motion.
